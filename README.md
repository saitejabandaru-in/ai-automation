## Core Pattern

Every automation follows this pattern:
1. **Trigger** — Schedule, webhook, file drop, or manual
2. **Fetch** — Get the data/document
3. **Process** — AI extraction, classification, or generation
4. **Format** — Structure the output
5. **Deliver** — Email, Slack, save to file, update DB

## Stack

- **Python** — Core automation logic
- **Anthropic/OpenAI APIs** — AI processing
- **pandas** — Data manipulation
- **Schedule/APScheduler** — Time-based triggers
- **smtplib** — Email delivery
- **requests** — API calls
